# Secure Quantum Key Distribution Framework for B5G Communication

## Overview
This repository provides a high-level overview of research on secure
Quantum Key Distribution (QKD) for B5G/6G communication systems.

The detailed algorithms, simulation framework, and experimental
implementation are not included in this repository due to
publication confidentiality.

## Research Status
This research work has been accepted for publication.

Full technical details and code will be released after the
official publication of the research paper.
